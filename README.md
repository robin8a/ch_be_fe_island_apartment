# Island Apartment — Amplify Gen 2 SPA

React + TypeScript + Vite frontend with an AWS Amplify Gen 2 backend (Cognito auth + AppSync/DynamoDB data).

## Prerequisites

- Node.js 18+
- AWS account and credentials configured (`aws configure` or environment variables)

## Local development

Use two terminals:

**Terminal 1 — Cloud sandbox** (provisions AWS resources and writes `amplify_outputs.json`):

```bash
npx ampx sandbox
```

**Terminal 2 — Frontend dev server:**

```bash
npm run dev
```

Open the URL Vite prints (usually `http://localhost:5173`). Sign up / sign in uses email via Cognito. The data layer includes a minimal `Placeholder` model (owner-scoped) until you define real domain models in `amplify/data/resource.ts`.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start Vite dev server |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |
| `npx ampx sandbox` | Local cloud sandbox (watch mode) |
| `npx ampx sandbox delete` | Tear down sandbox resources |

## Project layout

```
├── amplify/
│   ├── auth/resource.ts    # Cognito (email login)
│   ├── data/resource.ts    # AppSync schema & authorization
│   └── backend.ts          # Backend entry
├── src/
│   ├── main.tsx            # Amplify.configure(outputs)
│   └── App.tsx             # Authenticator + app shell
└── amplify_outputs.json    # Generated by sandbox (gitignored)
```

## Deploy via Git

1. Push this repository to GitHub, GitLab, or Bitbucket.
2. In the [AWS Amplify console](https://console.aws.amazon.com/amplify/), choose **Host web app** and connect the repo.
3. Amplify Gen 2 detects `amplify/` and runs `ampx pipeline-deploy` on push.

`amplify_outputs.json` is generated during CI build; do not commit it.
